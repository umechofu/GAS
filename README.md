# RSSをChatworkに投稿するためのGoogle Apps Script
## 概要
- 任意のRSSが更新されると、Google Apps Scriptを介してChatworkに新着の投稿をするプログラムです。

## 事前に必要なもの
- Chatwork API token
- Google Apps Script

## 連携イメージ
1. RSSのURLがRSSを取得
2. 取得したRSSをXMLへ変換
3. タイトル、URL、ディスクリプションを取得
4. 取得したデータがスプレットシートにあるデータよりも多い　かつ　記事タイトルがスプレットシートにないことを判定
5. ４の条件に当てはまる場合、スプレットシートにタイトル、URL、ディスクリプションを最新データが一番上の行になるように記載
6. チャットワークへタイトル、URLを通知

## 仕様
- 1分毎にGASを実行。
- RSSとスプレットシートデータに差分がある場合、チャットワークへ通知。
